Hello World!


I am Timothy Stockton, and I enjoy web development, machine learning, software engineering, data analysis, and all things music. This staticly hosted website is a portfolio made to display some of my work.


Some of these projects were made for Free Code Camp where I completed three certifications: Responsive Web Design, JavaScript Algorithms & Data Structures, and Front End Development Libraries. Others were made while I was at Keene State College which is where I got my Bachelors degree (Computer Science major and a Mathematics minor).


The first full-stack application I worked on was a college research project called Owl Assist Information System (OAIS). I was one of the first 3 student devs, and I assisted in it's construction and was tasked with maintaining it. It is a Flask application that utilizes web scrapers, Dialog Flow, Google Cloud, AWS, Selenium, and more. This project was my first real exposure to Git and Github as well as Trello. There were a couple articles written about OAIS. The first was in The Equinox: April 14, 2021, and the second was in KSC Stories: April 16, 2021. There was also an Equinox Video Interview: April 29, 2021. Ultimately, I left the project because the school cut funding.
